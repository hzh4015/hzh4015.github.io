1.bitmap介绍

位图不是真正的数据类型,它是定义在字符串类型中,一个字符串类型的值最多能存储512mb字节的内容位上限: 2^(9(512)+10(1024)+10(1024)+3(8b=1B))=2^32b

2.setbit设置某一位上的值

语法：SETBIT key offset value （offset位偏移量，从0开始）
